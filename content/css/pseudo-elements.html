Pseudo-elements let you style specific parts of an element, or insert decorative content before and after it — without adding extra HTML. They're written with a double colon: ::before, ::after.

Every Netflix movie card has a gradient that fades from transparent at the top to dark at the bottom. That's not a separate HTML element — it's created with CSS using ::before or ::after. No extra divs needed.

::before and ::after

These pseudo-elements insert a virtual element before or after the content of an element. They must have a content property to appear — even if it's empty.

.movie-card::after {
  content: "";              /* required — can be empty */
  position: absolute;
  inset: 0;                 /* shorthand for top/right/bottom/left: 0 */
  background: linear-gradient(
    to top,
    rgba(0, 0, 0, 0.85) 0%,
    transparent 60%
  );
}

Combined with position: absolute, this creates a gradient layer that sits on top of the movie thumbnail — without any extra HTML.

Pseudo-classes: :hover, :nth-child

Pseudo-classes target elements in a specific state or position. Unlike pseudo-elements, they use a single colon.

/* Applies when the user hovers */
.movie-card:hover {
  transform: scale(1.08);
}

/* Targets every other card */
.movie-card:nth-child(even) {
  opacity: 0.9;
}

/* Targets only the first card */
.movie-card:first-child {
  border-left: 3px solid #e50914;
}

/* Targets only the last card */
.movie-card:last-child {
  border-right: 3px solid #e50914;
}

::placeholder

Style the placeholder text inside input fields — like the "Email address" hint on Netflix's form:

input::placeholder {
  color: rgba(255, 255, 255, 0.6);
  font-size: 14px;
}
