CSS Grid is a two-dimensional layout system that places elements into rows and columns at the same time. Where Flexbox handles a single row or column, Grid handles the full layout — perfect for card grids.

Flexbox lined up our movie row horizontally. But what if we want a proper grid of movie cards that wraps into multiple rows — like Netflix's "More to Watch" section? That's Grid's job.

display: grid

To create a grid, set display: grid on the parent container. Its direct children automatically become grid items.

.movie-grid {
  display: grid;
}

On its own, this doesn't do much. You also need to define the columns.

grid-template-columns

This property defines how many columns the grid has and how wide each one is.

/* Three equal columns */
.movie-grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
}

/* Same thing, shorter */
.movie-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}

The fr unit stands for "fraction." One fr takes up one equal share of the available space. Three columns of 1fr each gives you three equal columns.

gap

The gap property adds space between grid rows and columns — without needing margins:

.movie-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 12px;
}
💡

Click Try It Yourself to see the grid in action. Try changing the number in repeat(3, 1fr) to 2, 4, or 5 to see how the grid reflows.

auto-fill and minmax

The real power of Grid is creating a layout that automatically adjusts the number of columns based on available space. This makes your grid respond to any screen size without media queries.

.movie-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  gap: 8px;
}

Breaking this down:

  • auto-fill — create as many columns as will fit
  • minmax(200px, 1fr) — each column is at least 200px wide, but can grow to fill available space

Result: on a wide screen you get 5 or 6 columns. On a narrow screen you automatically get 2 or 3. No media queries needed.
