A CSS selector tells the browser which HTML elements to apply styles to. Tag selectors target every element of a type — class selectors let you target specific elements by name.

So far, every time we wrote h1 { color: white; }, it affected every single <h1> on the page. That's fine when the page is simple. But what happens when you have a Netflix logo in the top bar and another heading in the hero section? They can't both have the same styles.

That's why classes exist.

Tag Selectors

You've been using these already. A tag selector targets every element of that type on the page.

h1 {
  color: white;
}

p {
  font-size: 18px;
}

Class Selectors

A class lets you name a specific group of elements and target only those. You add a class to HTML with the class attribute, and target it in CSS with a dot (.).

In HTML:

<div class="hero">
  <h1>Netflix</h1>
  <p>Watch anywhere.</p>
</div>

In CSS:

.hero {
  background-color: black;
  padding: 60px;
  text-align: center;
}

The dot before hero is what tells CSS it's a class. Without the dot, CSS would look for an HTML tag called hero — which doesn't exist.

ID Selectors

An ID works like a class, but it should only be used once per page. You add it with the id attribute in HTML and target it with a hash (#) in CSS.

<div id="main-header">...</div>
#main-header {
  position: fixed;
  top: 0;
  width: 100%;
}
💡

Use classes for styling — they can be reused on multiple elements. Use IDs sparingly, and only when an element is truly unique on the page.

Grouping Selectors

If two selectors share the same styles, you can combine them with a comma to avoid repeating yourself:

/* Instead of writing the same thing twice... */
h1 { color: white; }
h2 { color: white; }

/* ...write it once */
h1, h2 {
  color: white;
}

Descendant Selectors

You can target elements that are inside a specific parent. This is useful when you want to style a heading inside the hero section differently from headings elsewhere.

/* Only h1 tags inside .hero */
.hero h1 {
  font-size: 48px;
}

/* Only p tags inside .top-bar */
.top-bar p {
  font-size: 14px;
}
