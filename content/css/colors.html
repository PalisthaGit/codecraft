CSS gives you multiple ways to define colors — hex codes, RGB, RGBA, and named colors. RGBA also lets you control transparency, which is essential for overlays and tinted backgrounds.

Hex Codes

Hex codes are the most common way to define colors in CSS. They start with # followed by six characters that represent the red, green, and blue values of a color.

button {
  background-color: #e50914;
}

That's Netflix red. Hex codes are popular because they're precise and universally recognized by browsers and designers.

Named Colors and RGB

You can also use basic color names or RGB values:

/* Named color */
button {
  color: white;
}

/* RGB — same result, using numbers 0–255 */
button {
  background-color: rgb(229, 9, 20);
}

All three formats work. Hex is the most common in production code.

transparent

You can also set a background to fully transparent — meaning it has no color at all and you can see whatever is behind it.

input[type="email"] {
  background-color: transparent;
}

This makes the email input blend into the dark background instead of showing a white box.

Background Images

You can also use an image as the background instead of a solid color. For that, we use background-image:

body {
  background-image: url('https://assets.nflxext.com/ffe/siteui/vlv3/9ba9f0e2-b246-47f4-bd1f-3e84c23a5db8/web/NP-en-20251020-TRIFECTA-perspective_774f89dc-b75d-4cfc-b1fe-52086741fae3_large.jpg');
}

Now the full background of the page is the Netflix hero image.

Making Text Readable Over an Image

After adding the background image, the text can become hard to read. We can fix this by adding a dark, semi-transparent layer over the image — like tinted glass over a photo.

To do that, we use RGBA colors.

RGBA

RGBA works like RGB but adds a fourth value — the alpha channel — which controls opacity:

/* rgba(red, green, blue, opacity) */
rgba(0, 0, 0, 0.75)
  • The first three numbers are red, green, and blue (0–255)
  • The last number is opacity — from 0 (fully transparent) to 1 (fully solid)

So rgba(0, 0, 0, 0.75) means black at 75% opacity. That's perfect for a dark tint over our background image.

Blending the Color and Image

To blend the dark color and the background image together, we use background-blend-mode:

body {
  background-color: rgba(0, 0, 0, 0.75);
  background-image: url('...');
  background-blend-mode: multiply;
}

Now the dark tint and the image blend together. The background still looks rich, but the white text is easy to read against it.
