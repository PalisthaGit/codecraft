Flexbox is a CSS layout mode that arranges elements in a row or column and lets you control how they're spaced and aligned. It's the most-used tool for building navbars, hero sections, and button groups.

Right now, our top bar has the Netflix logo and a Sign In button — but they both sit on the left, stacked on top of each other. Flexbox fixes that in three lines.

display: flex

To use Flexbox, set display: flex on the parent element. Its direct children automatically become flex items arranged in a row.

.top-bar {
  display: flex;
}

That's it. The logo and button immediately line up side by side.

justify-content

justify-content controls how items are spread along the main axis (horizontal in a row).

.top-bar {
  display: flex;
  justify-content: space-between;  /* logo far left, button far right */
}

Common values:

justify-content: flex-start;     /* pack to the left (default) */
justify-content: flex-end;       /* pack to the right */
justify-content: center;         /* center everything */
justify-content: space-between;  /* first at left edge, last at right edge */
justify-content: space-around;   /* equal space around each item */

align-items

align-items controls how items line up on the cross axis (vertical in a row).

.top-bar {
  display: flex;
  justify-content: space-between;
  align-items: center;   /* vertically centered */
}

Without align-items: center, items sit at the top of the container. With it, they sit perfectly in the middle — no matter their height.

Centering the Hero Content

Flexbox is also perfect for centering content inside the hero section — both horizontally and vertically. Set flex-direction: column to stack items vertically, then center them:

.hero {
  display: flex;
  flex-direction: column;
  justify-content: center;   /* center vertically (in column mode) */
  align-items: center;       /* center horizontally (in column mode) */
  min-height: 100vh;
}

gap

The gap property adds space between flex items — without needing margins:

.hero-form {
  display: flex;
  gap: 12px;
}

.top-bar {
  display: flex;
  gap: 20px;
}

Building the Movie Row

Flexbox also works for the movie row — the horizontal strip of thumbnails. Setting overflow-x: auto makes it scrollable when there are more cards than space.

.movie-row {
  display: flex;
  gap: 8px;
  overflow-x: auto;
  padding: 16px 40px;
}

.movie-card {
  flex-shrink: 0;     /* don't let cards shrink smaller than their set size */
  width: 200px;
  height: 120px;
  border-radius: 4px;
  overflow: hidden;
}
💡

flex-shrink: 0 prevents cards from getting squished. Without it, flex items shrink to fit the container and your cards lose their shape.
