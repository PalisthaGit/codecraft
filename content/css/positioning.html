CSS positioning controls where an element sits on the page. With position: fixed, elements stay in place as the user scrolls. With position: absolute, elements can be placed precisely inside a parent.

When you scroll down on Netflix, the black navbar stays at the top. The movie title text sits on top of the thumbnail image. Both of these are done with CSS positioning.

position: static

This is the default. Every element starts here — it just sits in the normal flow of the page, one after another.

/* You'll never actually write this — it's the default */
.element {
  position: static;
}

position: relative

relative lets you nudge an element from where it would normally sit, without affecting the layout around it. It also creates a positioning context for any absolute children inside it.

.movie-card {
  position: relative;   /* makes this the anchor for absolute children */
}

position: absolute

absolute removes an element from the normal flow and places it precisely inside its nearest relative (or fixed) parent using top, right, bottom, and left.

.movie-card {
  position: relative;
}

.movie-title {
  position: absolute;
  bottom: 8px;
  left: 8px;
  color: white;
  font-size: 14px;
}
💡

Click Try It Yourself to see text overlaid on a movie card using absolute positioning. Try changing bottom and left to move the text around.

position: fixed

fixed locks an element to the viewport — it stays in place as the user scrolls. This is exactly what Netflix uses for its navbar.

.top-bar {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;            /* stretches to full width */
  z-index: 100;        /* sits above all other content */
  background-color: rgba(0, 0, 0, 0.9);
}
💡

When you fix the navbar, the page content starts at the top behind it. Add padding-top to the body equal to the navbar height to prevent overlap.

z-index

z-index controls which element appears on top when elements overlap. Higher numbers sit in front.

.top-bar {
  position: fixed;
  z-index: 100;      /* always on top */
}

.hero {
  z-index: 1;        /* below the nav */
}

.modal {
  z-index: 999;      /* above everything */
}
💡

z-index only works on elements that have a position set (anything other than static).
